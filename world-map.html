KaiJu Girls Route Map Guide

KaiJu Girls does not need a giant physical map to have route complexity. The useful map is a story map: memory beats, KaijuTamer progression, companion scenes, CG unlocks, and save points that let players replay without spoiling themselves. This page is structured as a launch-ready route notebook, so it can be filled in when a demo or final build confirms route names.

Route Map Layers

  • Main story: amnesiac protagonist and recovered memory beats.
  • KaijuTamer path: scenes tied to taming and traveling with kaiju.
  • Companion notes: names, roles, scene context, and official screenshot references.
  • CG tracker: chapter, route, unlock condition, and voice-scene marker.

Save Path Strategy

Keep clean saves before major choices. Name notes by chapter and scene type, not late-story spoilers. A good route map protects first-time readers while still helping completion players.

CG List Structure

Steam highlights 130 CG images. The best future page is not a gallery dump; it is a checklist with missing/unlocked state, route dependency, scene type, and whether the scene has voice work.

What To Wait For

Do not invent route names, endings, or companion profiles before official confirmation. Use the beginner guide and PC specs while waiting for demo or launch details.

Suggested Route Notebook

Use a neutral notebook format until official chapter names are confirmed. The goal is to help completion players without spoiling first-time readers.

  • Segment: Opening, KaijuTamer setup, companion travel, memory recovery, route branch, or CG-focused scene.
  • Save point: Before choice, before route lock, before CG scene, or before late-story reveal.
  • CG status: Missing, unlocked, alternate version unknown, or final condition unverified.
  • Voice marker: Selected-scene voice present, no voice, or unverified.
  • Spoiler label: Safe, route, late story, or hidden.

Companion Documentation

Until names and route structures are public, do not force character profiles. Track companions by visible scene context, official screenshots, Steam copy, and whether the scene appears tied to taming, memory recovery, travel, or conflict. Once official names are available, this section can become a character-route index.

CG Completion Workflow

Start with a first-read save, then create route checkpoints before major choices. After each scene, record the CG label, scene type, save file, and whether returning later would spoil a new player. For a more detailed tracker, use the CG checklist.
